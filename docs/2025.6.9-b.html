Conclusion
  • When using Chi-square test to compare read counts difference between the groups with and without C34, there can be many extremely small p-values, some of them might be false positive.
  • I further tested using DESeq2 to compare read counts difference between the groups with and without C34. DESeq2 is originally intended for differential expression gene analysis, however, it can be a good tool if assuming 1) the total read count of the amplicon is an analogy of the sequencing depth of a transcriptome, 2) each position of the amplicon is an analogy of a gene, and 3) the mutated read count at each position of the amplicon is an analogy of the gene expression level.
  • DESeq2 uses Wald test and can accomodate replicates in the analysis, which seemed to give less extreme p-values and fewer false positive signals comparing to the plain Chi-square test I did.
  • C12 seemed to consistently detect C34’s binding sites within the SL5 sequence (the position numbers are matched betweeen the Arc plot and volcano/manhattan plots). C30’s signals seemed to be more noisy and less significant.
